Parts I Used

Fatter Matter uses a small set of simple components that work together to scan items and display information.

These parts include the microcontroller, the screen, the barcode scanner, and the wiring that connects everything.

Each component plays an important role in making the device function smoothly and reliably.

By combining these basic parts, the project shows how easy it can be to build a working system that solves a real problem.

  • • Display Screen
  • • Barcode Scanner

  • • Power Source
